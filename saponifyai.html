Overview

Saponify AI combines the precision of a professional soap calculator with the conversational ease of AI. It calculates safe, accurate recipes using verified saponification values, validates formulations for common issues, and guides users through the craft of soap making - from choosing oils to troubleshooting problems. Built as a demonstration of RAG-enhanced AI assistants, it shows how domain-specific knowledge can be seamlessly integrated with large language models.

Technology Stack

  • AI Models: Gemini 2.5 Flash (conversational AI), Claude Code (development)
  • Core Engine: Custom soap calculator with verified SAP values for 19+ oils
  • Architecture: RAG (Retrieval-Augmented Generation) with local knowledge bank
  • Features: Recipe validation, safety checks, retry logic with exponential backoff
  • Frontend: Vanilla JavaScript, CSS animations, localStorage persistence

Impact & Results

What started as an experiment became a fully functional tool. Saponify AI can calculate any soap recipe a user requests, warn about potential issues (like too much coconut oil), and explain the chemistry behind saponification - all through natural conversation. It demonstrates how AI assistants become truly useful when paired with domain-specific calculators and validated data.
