Overview

AI Chronicles is a living knowledge graph that maps the AI landscape in real-time. Every day, it automatically scrapes 100+ articles from arXiv research papers, Hacker News discussions, and leading AI company blogs. Using Gemini AI, it extracts entities (organizations, models, topics) and builds a force-directed graph showing how ideas, companies, and technologies connect. Explore trending topics, discover relationships between research papers, and stay current on AI developments‚Äîall updated automatically via GitHub Actions.

Key Features

Automated Aggregation

Scrapes 8+ RSS feeds daily: arXiv (cs.AI, cs.LG, cs.CL), OpenAI, Google AI, Hugging Face, Anthropic, and MIT Tech Review. Also queries Hacker News API for trending AI discussions.

AI Summarization

Gemini 2.5 Flash generates concise summaries for articles. Entity extraction identifies 20+ AI organizations, 20+ models, and 13 topic categories from article text.

Knowledge Graph

HTML5 Canvas renders a force-directed physics simulation with 150+ nodes and 200+ edges. Four node types (articles, topics, orgs, models) are color-coded and sized by importance.

Trend Detection

Nodes are scored by recency, Hacker News points, and connection count. The graph intelligently displays the top 50 most relevant nodes, highlighting trending topics.

Technology Stack

  • Data Pipeline: Python scraper with feedparser for RSS, requests for APIs, and intelligent entity extraction
  • AI Integration: Gemini 2.5 Flash API for article summarization (optional, works without it)
  • Visualization: HTML5 Canvas with custom force-directed physics (repulsion, attraction, damping, gravity)
  • Automation: GitHub Actions runs daily at 6 AM UTC, auto-commits updates to the knowledge graph
  • Data Sources: arXiv RSS (3 feeds), Hacker News Algolia API, 5 AI company blog feeds
  • Frontend: Vanilla JavaScript with pixel-art UI, responsive grid layout, real-time stats
